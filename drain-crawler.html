Robotic Drain Crawler

Overview

This project focused on the development of an intelligent robotic drain crawler equipped with real-time video capture and an AI-based defect detection system. Designed for underground pipeline inspection, the crawler automates the process of identifying cracks, blockages, and structural defects using a Convolutional Neural Network (CNN), reducing reliance on manual inspection and improving efficiency.

Objectives

  • Design a mobile robotic platform for remote inspection of drainage pipelines.
  • Integrate a camera system for live video capture in low-light conditions.
  • Apply computer vision techniques to automatically detect anomalies using machine learning.
  • Collect and analyze inspection data to support predictive maintenance.

Approach

The crawler was equipped with a waterproof IR camera, ultrasonic sensors, and track-based mobility for rough terrain. Video frames captured during operation were processed both in real time and offline to detect issues.

Key components included:

  • Mechanical and Embedded Design: Custom-built chassis with Arduino-based motor control and sensor integration for autonomous navigation.
  • Computer Vision: A CNN model trained on a labeled dataset of pipeline images to classify defects such as cracks, corrosion, and blockages.
  • Data Processing: Captured images and metadata were stored and processed using Python, OpenCV, and NumPy.
  • Model Training and Evaluation: The CNN model achieved over 85% accuracy using a combination of data augmentation, transfer learning, and hyperparameter tuning.
  • Analytics: Defect trends were analyzed using Pandas, and insights were visualized using Matplotlib and Seaborn.

Impact

The crawler reduces manual labor and safety risks associated with underground inspection, while the AI system enables faster, more accurate identification of pipeline issues. This enhances preventive maintenance workflows and improves infrastructure longevity.

Skills & Tools Used

  • Deep Learning (CNN, Transfer Learning)
  • Computer Vision (OpenCV, Image Preprocessing)
  • Data Science (Python, Pandas, NumPy, Matplotlib, Seaborn)
  • Embedded Systems (Arduino, IR Camera, Ultrasonic Sensors)
  • Mechanical Design and Real-time Systems
